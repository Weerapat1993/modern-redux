# Modern Redux

## How to Install

1. Add dependency `modern-redux` into `package.json`
```json
{
  "dependencies": {
    "modern-redux": "https://github.com/Weerapat1993/modern-redux.git",
  }
}
```

2. Install Module
```sh
npm install
```
